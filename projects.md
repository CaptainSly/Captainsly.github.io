# The Project Page


## [TheGildedLibrary](https://github.com/CaptainSly/TheGildedLibrary)

A 3D RPG Focused Game Engine written in Java 25, utilizing LWJGL3, SDL3, GraalVM, and supports both OpenGL and Vulkan. The engine has an "Eat's its own cooking" philosophy. Anything the engine does, a plugin can do too!
